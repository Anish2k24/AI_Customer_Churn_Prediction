# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of AI Customer Churn Predictor
- Interactive Streamlit dashboard with 4 tabs
- Machine learning models: Random Forest and Gradient Boosting
- Real-time churn prediction with risk assessment
- Feature importance visualization
- Data explorer with CSV download
- Comprehensive documentation (README, CONTRIBUTING, CODE_OF_CONDUCT)
- MIT License

### Features
- 9 customer behavioral metrics input
- Risk level classification (CRITICAL/HIGH/MEDIUM/LOW)
- Personalized recommendations based on risk factors
- Model performance metrics display
- Feature importance analysis
- Dataset visualization and exploration

### Technical
- Python 3.8+ compatibility
- Scikit-learn for machine learning
- Streamlit for web interface
- Plotly for interactive charts
- Joblib for model persistence
- Comprehensive requirements.txt
